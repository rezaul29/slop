\subsection{Key Findings}

\begin{itemize}
    \item \textbf{ConvNeXt V2 + DINOv2} demonstrates exceptional performance with 96.26\% test accuracy and minimal overfitting (training accuracy: 100\%, test accuracy: 96.26\%).
    \item \textbf{DenseNet-201} achieves strong results with 95.42\% test accuracy and good generalization (training: 99.63\%, test: 95.42\%).
    \item \textbf{Swin Transformer V2} shows competitive performance with 95.16\% test accuracy and minimal overfitting (training: 99.61\%, test: 95.16\%).
    \item \textbf{VGG-19} achieves solid results with 93.11\% test accuracy, showing good generalization.
    \item \textbf{ResNet-101} shows the lowest performance at 88.58\% test accuracy, with the highest validation and test loss.
    \item All models achieve near-perfect training accuracy, indicating sufficient model capacity.
    \item The generalization gap (train-test difference) is smallest for ConvNeXt V2 + DINOv2, followed closely by DenseNet-201 and Swin Transformer V2.
\end{itemize}
